Чтобы приложение заново запускалось после закрытия, необходимо поместить его в планировщик задач:
1. Открыть "Планировщик задач" Win+R taskschd.msc
2. Создать новую задачу
3. Вкладка "Общие":
* "Имя" - Techno.AisIntegration.App
* "Описание" - Background application for integration between AISTPS and ASN via OPC-server
* Выполнять только для пользователей, вошедших в ситсему
* Выполнить с наивысшыми правами - Да

4. Вкладка "Триггеры":
Создать триггер 
	"При входе в систему"
	Повторять каждые 5 минут, бесконечно; 
	Выполнить

Создать триггер
	"Ежедневно"
	Начать - Текущее время
	Повторять - 1 день.
	Повторять каждые 5 минут, бесконечно;
	Выполнить

5. Действия
Создать "Запуск программы"
	Путь - С:\Program Files\TechnoAisIntegration\TehcnoAisIntegration.App.exe
